
## Angular CLI

- **What is Angular CLI?**  
  Angular CLI is a command-line tool for creating, building, and maintaining Angular applications.  
  Example:

  ```bash
  ng new my-app
  ```

- **Common CLI Commands**  
  Examples include `ng serve`, `ng build`, and `ng generate`.  
  Example:
  ```bash
  ng generate component my-component
  ```
